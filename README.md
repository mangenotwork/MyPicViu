# MyPicViu
我的图片查看器，pc软件，查看图片详细信息，显示颜色分布，图像处理等功能，图片文件分类管理，主打支持linux系统的桌面软件。


#### case

1. [ok] 从打开目录或图片文件到图片正常显示

2. [ok] 图片显示与图片信息的展示与图片色彩分布的展示

3. 图片操作界面与图片操作交互

4. 图片文件拷贝与转移

5. 图片目录批量操作有批量命名批量加水印批量缩略图

6. 相册管理系统可指定目录创建相册进行图片分类存照不同的目录里
